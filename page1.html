Guide Video Thumbnail A teaser advertisement for an online video guide. Designed using Adobe Photoshop, the work contains assets extracted using a frame analyzer. This was used to separate multiple on screen elements from an active 3D software, and import them to isolate these 3D objects and manipulate them in a 2D design software. I approached this project with a goal of establishing a brand identity with cool, soft, green colors and a certain tone of calmness and professionalism.
